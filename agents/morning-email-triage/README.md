# morning-email-triage

This agent analyzes your Gmail inbox to find important unread emails and transform them into tasks.

## Features
- **Smart Filtering**: Automatically ignores newsletters and spam.
- **Automated Workflows**: Uses `gws workflow +email-to-task` for efficiency.
- **Safe**: Only reads emails; never deletes or archives.

## Usage
Simply ask Antigravity to "triage my emails" or "run the morning email triage".
